Data Analyst Internship - Task 1

Dataset: Netflix Movies and TV Shows

Objective:
To clean and standardize a real-world messy dataset using Excel.

Steps Performed:
- Imported raw Kaggle dataset and preserved it as Raw_Data.xlsx
- Identified and highlighted missing values
- Removed duplicate records using key columns
- Standardized text fields (title, director, cast, country)
- Validated date and numeric formats
- Added Data_Quality_Notes for data quality documentation
- Created final cleaned dataset

Final Deliverables:
- Raw_Data.xlsx
- Cleaned_dataset.xlsx
- cleaned_dataset.csv

Result:
Clean structured dataset ready for analysis and reporting.
